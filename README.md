# Calculadora de Desconto

Veja rapidamente o preço final de um produto após aplicar um desconto percentual. O programa pede o preço original e a porcentagem, faz o cálculo e mostra o valor atualizado no terminal.

## Como funciona

O cálculo aplicado é:

```text
desconto = preço × (porcentagem ÷ 100)
preço final = preço − desconto
```

Depois de mostrar o resultado, o programa permite realizar uma nova conta.

## Requisitos

- Python 3
- terminal no Windows ou Linux

## Como executar

```bash
git clone https://github.com/riique/Porcentagem-Desconto.git
cd Porcentagem-Desconto
python Percent-Descont.py
```

Digite `1` para começar, informe o preço e a porcentagem de desconto e leia o resultado formatado em reais.
